## 6. AI 분석 및 추천 알고리즘 엔진 Spec (AI & Recommendation Engine)

### 6.1 짬뽕 특화 11개 항목 평가 매트릭스
기존 축제용 Rule Scorer를 대체하는 짬뽕 전용 형태소/키워드 매핑 테이블 데이터셋 레이어입니다.

| 분석 항목 | 형태소 탐색 핵심 키워드 예시 | 점수 산출 가이드라인 | 유저 페이지 노출 텍스트 예시 |
| :--- | :--- | :--- | :--- |
| **1. 국물 맛** | 진하다, 깊다, 깔끔하다, 시원하다, 싱겁다, 텁텁하다 | 긍정어/부정어 가중 비율 매핑 (0~100) | "국물이 묵직하고 깊은 맛이 일품입니다." |
| **2. 매운맛** | 맵다, 칼칼하다, 얼큰하다, 땀난다, 순하다, 밋밋하다 | 빈도수 및 부사어 조합 분석 기반 1~5단계 표준화 | "신라면보다 매콤해 해장용으로 좋습니다." |
| **3. 불향** | 불맛, 불향, 웍질, 탄내, 그을린, 탄 향 | 불향 관련 형태소 밀도 및 만족도 연동 (0~100) | "입안 가득 퍼지는 강렬한 웍 불향이 특징입니다." |
| **4. 면 식감** | 쫄깃하다, 탱탱하다, 퍼졌다, 밀가루 냄새, 부드럽다 | 식감 형용사 추출 가중치 (0~100) | "수타면처럼 쫄깃함이 오래 유지되는 면발입니다." |
| **5. 해산물/고명**| 신선하다, 차돌, 오징어 가득, 홍합, 실하다, 부실하다 | 고명 종류 분류 및 신선도/양 만족도 복합 스코어링 | "씨알 굵은 동죽과 오징어가 풍부합니다." |
| **6. 양** | 배부르다, 양 많다, 푸짐하다, 적다, 곱빼기 같은 | 분량 관련 명사/형용사 밀도 스코어링 | "일반 사이즈도 곱빼기 수준으로 푸짐합니다." |
| **7. 가격 만족도**| 가성비, 비싸다, 값어치, 저렴하다, 이 돈이면 | 가격 수치 데이터와 결합된 감성어 매칭 | "만원이라는 가격이 아깝지 않은 구성입니다." |
| **8. 웨이팅** | 웨이팅, 줄 서다, 오픈런, 극악, 바로 입장, 회전율 | 대기 시간 수치 패턴 및 피로도 감성어 추출 | "주말 피크 타임 기준 최소 1시간 대기 발생." |
| **9. 위생** | 깔끔하다, 청결, 노포라 더럽다, 끈적하다, 위생 관리 | 환경 부정어 검출 시 감점 가중치 적용 | "테이블과 양념통이 청결하게 유지됩니다." |

### 6.2 다차원 가중치 추천 스코어링 공식 (의사결정 1 반영)

$$Score_{total} = (Score_{blog\_ai} \times 0.8) + (Score_{youtube\_ai} \times 0.2)$$

* **$Score_{blog\_ai}$ 계산 공식**:
  $$Score_{blog\_ai} = \frac{\sum_{i=1}^{n} (W_i \times S_i)}{\sum_{i=1}^{n} W_i} - Penalty_{waiting}$$
  * $W_i$ (사용자 가중치): 유저가 선택한 핵심 선호 항목은 고가중치(2.0), 선택하지 않은 기본 항목은 1.0 적용.
  * $S_i$: 0~100점 사이로 정규화된 맛집의 항목별 AI 분석 스코어.
  * $Penalty_{waiting}$: 유저가 '웨이팅 적은 곳'을 원했으나 해당 업소의 대기 지수가 높을 경우 패널티 감점 적용.
* **$Score_{youtube\_ai}$ 계산 및 예외 처리 (Fallback)**:
  * 유튜버 채널의 영상 자막 데이터와 댓글 데이터를 동일한 11개 항목 매트릭스로 분석하여 0~100점 사이로 정규화.
  * **유튜버 미방문 업소 처리**: $Score_{youtube\_ai}$ 정보가 없는 업소는 블로그 AI 점수($Score_{blog\_ai}$)를 **100% 비중**으로 대체 적용하되, 유튜버 인증 배지 및 유튜버 가산점 혜택은 제공되지 않음.
